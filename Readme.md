# D&D Session Manager (Local)

Локальний менеджер кампаній і сесій для Dungeons & Dragons. Дозволяє створювати кампанії, сесії та планувати кожну сесію через інтерактивний To‑Do шаблон. Усі дані зберігаються локально у вигляді JSON-файлів.

---

## Можливості

- Кампанії
  - Створення
  - Перейменування
  - Позначення як завершеної / активної
  - Видалення

- Сесії
  - Створення (назва або автоматично YYYY-MM-DD)
  - Перейменування
  - Позначення як завершеної / активної
  - Видалення
  - Автозбереження при будь-якій зміні

- Планування сесії
  - Ядро сесії
  - Сцени (можна додавати, видаляти, згортати)
  - NPC та фракції
  - Контент сесії
  - Результат сесії
  - Прогрес підготовки

- Дані
  - Кожна кампанія — окрема папка
  - Кожна сесія — окремий JSON файл
  - Легко робити резервні копії

---

## Встановлення

1. Встановити Node.js
   
   https://nodejs.org/

2. Відкрити папку проєкту в терміналі та виконати:

```bash
npm install
```

---

## Запуск

### Варіант 1 — через bat файл

Просто запусти:

```
run_project.bat
```

Він:
- запустить Node сервер
- відкриє сайт у браузері

Можна створити ярлик на робочому столі.

## Де зберігаються дані

Усі дані лежать у папці:

```
data/campaigns/
```

Кожна кампанія — окрема папка.
Кожна сесія — окремий JSON файл.

Це означає, що можна:
- копіювати кампанії
- робити backup
- редагувати JSON вручну
